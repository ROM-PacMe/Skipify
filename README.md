# SpotifySkip

SpotifySkip lets you enable tablet-only Spotify freemium features on mobile devices.

## Features
- Unlimited skips
- On-demand songs
- Disable shuffle mode

## Future
- Be Compatibility with 8.x Spotify Versions.
